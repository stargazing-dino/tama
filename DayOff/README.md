# DayOff/ — room theme & prop assets

This folder is intentionally empty in the repo. Contents are purchased
itch.io assets that cannot be redistributed.

## Setup

Download the pack: https://misterdll.itch.io/dayoff

Unzip it so the pack's top-level folders (`Car/`, `Characters/`,
`Kitty/`, `Objects/`, `Tileset/`, `License.txt`, `Readme.txt`) land
directly in this directory.

The pygame preview and firmware reference specific files from `Objects/`
and the themes are sliced out into `firmware/assets/themes/`. See
`pygame/main.py` for the exact paths consumed.

See `ATTRIBUTIONS.md` at the repo root for credits and terms.
